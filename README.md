# MINIJOGOS

- MINIJOGOS é um programa desenvolvido em Linguagem C, o qual reúne 3 minijogos em um menu interativo. O jogador escolhe qual jogo deseja jogar.
- Tal projeto foi desenvolvido como um trabalho prático da disciplina deAlgoritimo e Codificação de Sistemas.
  
- Dentre os jogos temos:
1. Perguntas e Respostas 
  - Quiz múltipla escolha.
  - Aqui você terá 5 perguntas e 5 alternativas.
  - Ao final delas, você terá o resultado das suas respostas.

2. Cobra na Caixa
  - Jogo de sorte, em que o jogador deve encontrar o botão e torcer para não encontrar a cobra dentro da caixa.
  - O jogador deve escolher uma "persona" dentre as 7 listadas.
  - O Jogador que iniciará será sorteado pelo próprio programa.
  - Caso o Jogador encontre a COBRA, o jogo acaba e vence quem não a encontrou. Se o Jogador achar o botão, quem achou ganha. E se ninguém achar o jogo continua rodando.

3. Gousmas War
 - Jogo de estratégia.
 - Cada jogador controla duas gousmas, que começam com 1 de fúria.
 - A cada turno o jogador pode escolher entre atacar o adversário ou dividir suas fúrias entre as gousmas.
 - O Jogador escolhe qual gousma deseja utilizar, seja a (1) ou (2).
 - Caso escolha atacar, a fúria total da gousma atcante é adicionada na gousma atacada.
 - Caso ele escolha dividir, o Jogador escolherá quanto deseja passar para a outra (desde que nao zere a fúria da gousma).
 - Uma gousma é eliminada assim que atinge mais que 5 de fúria.
 - Perde quem tiver suas duas gousmas eliminadas.
