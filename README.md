# minijogos

- MINIJOGOS é um programa desenvolvido em Linguagem C, o qual reúne 3 minijogos em um menu interativo. O jogador escolhe qual jogo deseja jogar.
- Tal projeto foi desenvolvido como um trabalho prático da disciplina deAlgoritimo e Codificação de Sistemas.
  
- Dentre os jogos temos:
1. Perguntas e Respostas - Quiz múltipla escolha.
2. Cobra na Caixa - Jogo de sorte, em que o jogador deve encontrar o botão e torcer para não encontrar a cobra dentro da caixa.
3. Gousmas War - Jogo de estratégia. Cada jogador controla duas gousmas, que começam com 1 de fúria. A cada turno o jogador pode escolher entre atacar o adversário ou dividir suas fúrias entre as gousmas. Uma gousma é eliminada assim que atinge mais que 5 de fúria. Perde quem tiver suas duas gousmas eiminadas.
